class mhe.EstimatorMHE.EstimatorMHE(t_step, setup_mhe)[source]

Bases: object

Class representing the MHE estimator. It contains the model instance whose parameters are to be estimated.

This class defines the parameters of the estimator and is capable of executing the estimation loop.
