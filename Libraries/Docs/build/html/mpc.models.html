mpc.models.ToggleSwitchModel module

class mpc.models.ToggleSwitchModel.ToggleSwitchModel(stochasticity, LacI_ref, TetR_ref, t_step, total_time, avg_period, *args)[source]

Bases: object

Class representing the Toggle Switch model used by the MPC controller. This defines the model equations, the cost function and constraints to be taken into account when applying the control strategy, and a set of methods for formatting, storing, exporting and plotting closed-loop simulation results and performance evaluation.
